# **App Name**: WasteWise AI

## Core Features:

- Image Analysis: Analyze uploaded images of leftover food to identify types and estimate quantities wasted, acting as a tool to classify waste.
- Waste Data Display: Present analyzed waste data visually to the user.
- Data Persistence: Data pertaining to uploaded images is stored locally in the browser's memory.
- Report Generation: Generate downloadable reports summarizing food waste analytics for specified periods.
- Tips Display: Provide users with tips on reducing food waste through a dedicated section.
- Contact Form: Allow users to send messages.

## Style Guidelines:

- Primary color: Forest green (#38A3A5), reflecting freshness and sustainability.
- Background color: Very light green (#F0FAF7), creates a clean, airy feel.
- Accent color: Teal (#47CACC), to highlight key data and calls to action.
- Body and headline font: 'PT Sans', a humanist sans-serif providing a modern look with some warmth and personality.
- Use minimalist, clear icons representing different types of food and waste metrics.
- Implement a clean, grid-based layout to ensure responsiveness and ease of navigation.
- Incorporate subtle transitions and loading animations to improve user experience.
